SOCIAL Media Web Application

Java-based social networking platform

Project Overview

I spearheaded the development of a dynamic social networking web application that prioritized user engagement and interactivity. Leveraging a robust tech stack comprising Java, Microsoft SQL, CSS, JSP, and Servlet technologies, I meticulously structured the codebase following the MVC (Model-View-Controller) pattern. The application was hosted on Apache Tomcat, making extensive use of JSP servlets for seamless communication between the front-end and back-end.

Notable features included user registration and authentication, which seamlessly facilitated friend requests, post sharing, and real-time messaging. This project aimed to cultivate meaningful user interactions while providing robust privacy settings, giving users full control over their data and interactions. In essence, it showcased my adeptness in combining Tomcat Apache, JSP servlets, CSS, Java, and Microsoft SQL to create an engaging and privacy-conscious social networking platform."
