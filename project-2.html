Garment Industry Web Application

C#-Based Inventory Management Platform

Project Overview

In this project, I harnessed my C# expertise for backend development, creating a suite of efficient pages and features. To ensure both dynamic and static workflows, I implemented data structures such as DataTable, TreeView, and GridView. Additionally, I managed a comprehensive database hosted on Microsoft Server 2008 R2, comprising over 100 tables. Utilizing Visual Studio 2019, I actively debugged and reported workflow progress.

The project's centerpiece was the development of an inventory management web application, boasting a substantial codebase exceeding 14,000 lines. This transformation involved the conversion of a desktop application into a web-based solution, showcasing my adaptability and innovation in software development. Throughout, I took on a leadership role, fostering teamwork, effective communication, and the delivery of practical solutions. The entire project was developed using C# within the ASP.NET Framework, resulting in a sophisticated and highly functional inventory management system.
